# ЗАТОЧКА.ТСК

Сервисная мастерская в Томске: **заточка профессионального инструмента** и **ремонт аппаратов** для частных мастеров beauty (маникюр, парикмахеры, грумеры и т.д.). Это веб-система учёта заказов поверх Laravel-монолита.

## Для чего

Единая точка вместо разрозненных заявок, звонков и табличек: заявка с сайта → заказ в цехе → выдача клиенту, с прозрачностью для клиента и контролем для сотрудников.

## Акторы

| Актор | Канал | Роль |
|-------|-------|------|
| **Клиент** | Сайт, `/api/*`, личный кабинет | Заявка, профиль, история заказов, отзыв после выдачи |
| **Мастер** | POS (`/api/pos/*`) | Свои заказы в цехе: в работу, работы, готовность |
| **Менеджер** | Админка Filament (`/cp`) | Создание заказов, цены, склад, выдача, документы |

## Цель

Убрать хаос в операционке мастерской: один поток данных от обращения клиента до закрытого заказа.

## Какие вопросы решает

- Как принять заявку с сайта и не потерять её до цеха
- Кто и на каком этапе ведёт заказ (менеджер → мастер → выдача)
- Сколько стоит работа и что уже сделано
- Где клиент видит свои заказы без лишней «кухни» статусов
- Учёт материалов, оборудования и базовых документов (PDF)

Доставка — опция к заказу (флаг), не отдельная услуга. Один филиал, B2C.
