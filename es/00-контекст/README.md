# 00 — Бизнес-контекст и границы

## Зачем этот блок

Зафиксировать, **что за бизнес**, для кого, где границы системы и что сознательно **не** входит в scope.

## Опросник (группа 1) — ✅ завершён

### 1.1 Суть бизнеса

| Вопрос | Ответ |
|--------|-------|
| Одним предложением: чем занимается компания? | Сервисная мастерская в Томске: заточка профессионального инструмента и ремонт аппаратов для частных мастеров beauty/grooming-сферы |
| Основные услуги (перечень) | Заточка профессионального инструмента; ремонт аппаратов/оборудования |
| Дополнительные услуги (если есть) | Курьерская доставка — **опция** к заказу на заточку/ремонт (забрать/привезти), не отдельный тип услуги |

### 1.2 Клиенты и рынок

| Вопрос | Ответ |
|--------|-------|
| Целевая аудитория | B2C: частные мастера (маникюр, парикмахеры, грумеры, барберы и т.д.) |
| География (город, регион, доставка) | Только Томск: офлайн-точка + курьер по городу |
| B2C / B2B / смешанная модель | B2C (частные мастера) |

**Каналы привлечения:** сайт, звонок/мессенджер, личный визит, курьер забирает инструмент, сарафан, соцсети.

### 1.3 Границы системы

| Вопрос | Ответ |
|--------|-------|
| Что система **должна** делать (MVP) | Онлайн-заявки; ЛК клиента; POS мастера; склад; документы (PDF); админка менеджера (Filament) |
| Что система **не** делает (вне scope) | Бухгалтерия/1С; зарплата; маркетинговая CRM; закупки у поставщиков; франшиза/мультигород; онлайн-оплата |
| Смежные системы | Бухгалтерия ведётся отдельно, интеграция не планируется |
| Филиалы | Одна точка; мультивыбор филиала не нужен |

**Оплата в реальности (в системе не ведётся):** наличные при выдаче/курьере, карта на терминале, безнал/перевод, гарантийный ремонт без оплаты.

### 1.4 Цели продукта (цифровая часть)

| Вопрос | Ответ |
|--------|-------|
| Зачем нужен софт (главная цель) | Убрать хаос: Excel/тетрадки/звонки → единая система заказов |
| KPI / метрики успеха (если есть) | _не зафиксированы_ |
| MVP vs «когда-нибудь» | MVP: заявки, ЛК, POS, склад, документы, админка. Позже: уведомления, бонусы, аналитика, онлайн-оплата |
