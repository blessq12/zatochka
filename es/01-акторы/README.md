# 01 — Акторы и роли

## Опросник (группа 2) — ✅ завершён

### Карта акторов

| Роль | Тип | Канал | Связь с другими |
|------|-----|-------|-----------------|
| **Клиент** | человек | Сайт (заявка без регистрации, ЛК), офлайн, телефон, соцсети | Заказ создаётся чаще сотрудником; клиент может сам с сайта |
| **Мастер** | человек | POS `/pos` | Всегда отдельный человек от менеджера |
| **Менеджер** | человек | Filament `/cp` | Создаёт заказы, выдаёт, материалы, склад, документы |

### Клиент

| Поле | Значение |
|------|----------|
| Кто | Частный мастер beauty/grooming (B2C) |
| Цель | Сдать инструмент/аппарат, получить обратно, видеть статус |
| Регистрация | Не обязательна для заявки |
| Действия в ЛК | Профиль, активные заказы, история, отзыв после выдачи, Telegram |
| Не делает | Не меняет статус заказа; не отменяет (в текущих ответах) |

### Мастер (POS)

| Действие | Да/нет |
|----------|--------|
| Взять заказ в работу | ✅ |
| Менять статус (в работе / ждём запчасти / готов) | ✅ |
| Добавлять/удалять работы | ✅ |
| Смотреть материалы на заказе | ✅ |
| Добавлять материалы | ❌ (только менеджер) |
| Внутренние заметки | ✅ |
| Создавать заказ | ❌ |
| Выдавать заказ | ❌ |
| Поиск оборудования / история ремонтов | ✅ |

### Менеджер (админка)

| Действие | Да/нет |
|----------|--------|
| Создание заказа от имени клиента | ✅ (основной канал) |
| Назначение мастера на заказ | ✅ (до взятия в работу) |
| Прайс | ✅ |
| Материалы/запчасти на заказ | ✅ |
| Документы (акты, накладные) | ✅ |
| Приход на склад, номенклатура | ✅ |
| Отмена заказа | ✅ |
| Выдача заказа («выдан») | ✅ |
| Пользователи (мастера, доступы) | ✅ |
| Модерация отзывов | ✅ |
