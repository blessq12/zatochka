# 11 — Глоссарий

| Термин | Определение | Синонимы | Не путать с |
|--------|-------------|----------|-------------|
| **Заявка (Lead)** | Запись с сайта до создания заказа; флаг `converted` | lead, обращение с сайта | Заказ |
| **Заказ (Order)** | Единица работы мастерской; воронка статусов | order | Заявка |
| **Работа (Work)** | Выполненная операция в цехе; цену ставит менеджер | work, работа по заказу | Work (мастер как User) |
| **Мастер** | Сотрудник цеха, POS | исполнитель | Клиент-мастер (beauty) |
| **Менеджер** | Админ Filament; создаёт заказы, выдаёт, склад | администратор | |
| **Клиент** | Частный мастер beauty — заказчик услуги | | Мастер (сотрудник) |
| **Снимок клиента (client_snapshot)** | Имя/телефон на заказе при отсутствии client_id | | |
| **Оборудование (Equipment)** | Аппарат для ремонта в реестре с серийниками | аппарат | Tool (инструмент заточки) |
| **Инструмент (Tool)** | Позиция заточки внутри заказа | | Equipment |
| **Выдача (issued)** | Заказ передан клиенту; финальный статус | | ready (готов, но ещё не отдан) |
| **Активный заказ (ЛК)** | Не issued и не cancelled; без отображения статуса цеха | | |
